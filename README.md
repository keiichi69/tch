
# Tài Chính Học – QuizApp 404 Pro

## Cách chạy
1. Cài Node.js (nếu chưa có).
2. Mở terminal trong thư mục này.
3. Chạy:
   ```bash
   npm install
   npm start```
   
4. Truy cập http://localhost:3000 để mở ứng dụng.

## Mô tả
- Gồm 404 câu hỏi và đáp án được nhúng sẵn.
- Hai chế độ: **Luyện tập** (hiện đáp án đúng/sai ngay) và **Kiểm tra** (random câu hỏi, giới hạn thời gian).
- Tự động tính thời gian theo công thức 60 phút / 100 câu.
- Hiển thị tiến độ và kết quả chi tiết sau khi nộp.
